# 🚨 SOS: System of Support
**Access help. Anywhere. Anytime.**

SOS is an **offline-first community resource finder** that helps people quickly access
food, shelter, education, and emergency support — even without internet access.

---

## ✨ Features
- Offline-first resource discovery
- Retro terminal-style UI
- CLI mode (`search hospitals`, `list pune`)
- Emergency helplines always visible
- Micro Guides for first-time visits
- Privacy-first (local data only)
- Demo-ready dummy data

---

## 🛠️ Built With
- React
- TypeScript
- Vite
- Static JSON
- LocalStorage
- HTML & CSS
